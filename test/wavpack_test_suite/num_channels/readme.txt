
                        -------------------------
                        WAVPACK NUM CHANNELS TEST
                        -------------------------

The files in this folder test various channel configurations. Currently,
this is just mono (1 channel) and stereo (2 channels), but this will
eventually be expanded to include multichannel material. All files
contain md5 sums, so they can be easily verified.

Note that the stereo sample was selected for its obvious stereo effect.
